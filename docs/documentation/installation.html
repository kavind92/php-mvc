Quick Installation

Just extract downloaded .zip file into htdocs or www or html folder(public folder name based on server software)

/htdocs/
   mvc/
      mvc-system/
      my-application/
      public-files/
      index.php

Thats all. Now run

http://localhost/mvc/index.php

if you have online hosting then

http://domainname.domain/index.php

Single Installation

Directory Structure

/htdocs/

http://localhost/index.php

http://domainname.domain/index.php
